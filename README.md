Max.riddim is a bridge connecting Djay Pro and Max — inspired by the similar integration built by TouchDesigner.

Max.riddim was the winner project of the [Algoriddim](https://www.algoriddim.com) challenge of the 2026 Lisbon Music Hackathon by [Music HackSpace](https://musichackspace.org)

### Why Maxriddim?

| Without Maxriddim | With Maxriddim |
|---|---|
| Complex data communication setup | OSC connect & get formatted data easily |
| Technical barrier to entry | Accessible to beginner max users |
| Limited artistic options | Expanded creative possibilities |

Max.riddim is designed for **everyday users** who want to unlock richer artistic options native to max like dynamic surround, visuals and more — without wrestling with low-level data communication configuration.

Max.riddim handles all the complexity under the hood, so you can focus on what matters most — your art.

### Setup
1. Download folder into max packages folder
2. install ODOT from package manager inside Max
3. Run Max.riddim

*Maxriddim — less setup. More experimentation.*
